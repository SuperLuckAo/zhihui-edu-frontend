# C 端 - 智慧学院（用户端）

基于 **Vite + React 18 + TypeScript** 的 C 端（用户端）应用。

## 启动

```bash
# 在 monorepo 根目录
pnpm install
pnpm dev:c

# 或者在当前目录
pnpm dev
```

## 目录结构

```
c-end/
├── public/             # 静态资源
├── src/
│   ├── App.tsx         # 根组件
│   ├── main.tsx        # 入口
│   ├── index.css       # 全局样式
│   └── vite-env.d.ts
├── index.html
├── vite.config.ts
├── tsconfig.json
└── package.json
```
